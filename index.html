About This System
🛡️

What is this system?

An Intrusion Detection System (IDS) that analyzes network traffic features and classifies them as: normal traffic (BENIGN), distributed denial-of-service attack (DDoS), or port scanning (PortScan).

📊

What does the result mean?

The system displays probability scores for each class, interprets each feature by severity level, then recommends an IPS decision: Allow, Block, or Rate Limit.
