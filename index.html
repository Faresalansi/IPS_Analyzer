About This System
🛡️

What is this system?

An Intrusion Detection System (IDS) that analyzes network traffic features and classifies them as: normal traffic (BENIGN), distributed denial-of-service attack (DDoS), or port scanning (PortScan).

🎛️

Two Modes Available

Use Manual Input to set your own feature values using sliders and analyze any custom flow. Use Demo Mode to step through 30 pre-built labeled samples and track model accuracy.
